# UDP-RECEIVER

## Dependencies
   C++17, Cmake >= 3.10

## Build
   Run `build.sh` script which will compile the application and create the binary `udp_receiver`
   
   Type `./udp_receiver` for the help.

## Test
The application was tested under `UBUNTU-22.04-LTS`

Kernel version `6.5.0-18-generic`
